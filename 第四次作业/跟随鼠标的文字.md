# 跟随鼠标的文字

点击鼠标并移动鼠标，屏幕上会出现跟随鼠标的文字，它们大小不一，角度各异。文字大小受鼠标移动速度影响。按下 DELETE 键或 Backspace 键可清空屏幕。按下 UP 键和 DOWN键可改变文字旋转角度。按下‘ s （S）’ 键可将你作品存储到程序目录中一个名为goodworks的文件夹中。
